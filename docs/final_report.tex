\begin{abstract}
We present a comprehensive framework for predicting business success 
using the Yelp dataset. Our approach introduces novel user credibility 
weighting and temporal validation methods to address the unique challenges 
of business failure prediction. We achieve ROC-AUC of 0.80 using temporal 
validation, demonstrating realistic performance after correcting for data leakage.
\end{abstract}
